# Product Gap Analysis (Skill)

Purpose:
- Deliver clear, ranked product gap recommendations.

Process:
1. Identify 3-5 comparable tools or platforms.
2. Compare core flows and key capabilities.
3. List missing or weak features tied to user value.
4. Rank improvements by adoption/retention impact.

Constraints:
- Avoid speculative or vanity features.
- No feature bloat.

Output:
- Ranked recommendations only, with brief rationale.
